# GitHub setup (FrontEnd)

Remote: [varseeker/myassist-frontend](https://github.com/varseeker/myassist-frontend)

```bash
git remote -v
# origin  https://github.com/varseeker/myassist-frontend.git (fetch)
# origin  https://github.com/varseeker/myassist-frontend.git (push)
```

## Push pertama kali

```bash
git add .
git commit -m "Initial commit: MyAssist Web"
git push -u origin main
```

## Ubah remote (jika perlu)

```bash
git remote set-url origin https://github.com/varseeker/myassist-frontend.git
```
